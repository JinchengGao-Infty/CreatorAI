# CreatorAI v2

AI 辅助小说写作工具 - 桌面应用版

> ⚠️ **开发中** - 功能基本完成，正在完善中

## 功能

- 📝 **章节管理** - 新建/删除/重命名章节，自动保存
- 🤖 **AI 续写** - 生成正文 + 摘要，支持追加或替换
- 💬 **讨论模式** - 与 AI 探讨创作思路
- 📋 **摘要系统** - 自动生成并记录摘要
- 🎨 **文风预设** - 自定义写作风格，支持导入/导出
- ⚙️ **多模型支持** - 多端点管理，参数可调
- 🔐 **安全存储** - API Key 使用系统 Keychain 加密存储

## 技术栈

- **Tauri** - 轻量跨平台桌面框架
- **React + TypeScript** - 前端
- **Ant Design** - UI 组件库
- **Monaco Editor** - VS Code 编辑器核心
- **Rust** - 后端

## 界面风格

VS Code 风格布局 + 象牙白主题

```
┌──────┬────────────┬──────────────────┬─────────────────┐
│ 活动 │   侧边栏    │      编辑器       │    AI 助手      │
│  栏  │            │                  │                 │
│      │ - 章节目录  │   正文编辑区域    │  续写/讨论模式   │
│ 📁   │ - 摘要记录  │                  │                 │
│ 📋   │ - 文风预设  │                  │   对话历史      │
│ ⚙️   │ - 模型设置  │                  │                 │
└──────┴────────────┴──────────────────┴─────────────────┘
```

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建
npm run tauri build
```

## 环境要求

- Node.js 18+
- Rust 1.70+
- npm 或 pnpm
